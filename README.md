# COSC HackWeek 2026

Welcome to the COSC HackWeek 2026 project repository! This week-long hackathon features daily projects covering various aspects of modern software development, from web fundamentals to advanced AI and DevOps practices.

## 📅 Project Structure

The repository is organized by days, with each day focusing on different technologies and concepts:

### Day 1 - Web Development Fundamentals
- **CGPA-Calculator**: Academic performance tracking tool
- **Expense-Tracker**: Personal finance management
- **Kanban-Board**: Project management with Kanban methodology
- **Rock-Paper-Scissor**: Classic game implementation
- **Soundboard**: Interactive sound effects player
- **my-first-extension**: Browser extension development

### Day 2 - Advanced Web Applications
- **Color-Blindness-Simulator**: Visual accessibility tool
- **Digital-Business-Card-Generator**: Digital card creation
- **File-Metadata-Inspector**: File analysis tool
- **digital-time-capsule**: Time-based content storage
- **habit-streak-tracker**: Productivity and habit tracking
- **white-board**: Collaborative drawing application

### Day 3 - AI & Security Tools
- **A-tiny-programming-language**: Custom language interpreter
- **Kavach-COSC**: Security vulnerability detection
- **Kavach-tool**: Advanced code security scanning
- **chat-with-ollama**: Local AI chat integration
- **cortex-agent**: AI agent framework

### Day 4 - DevOps & Advanced Applications
- **Dockerizee-cosc**: Containerization best practices
- **Pikachumascot**: Interactive mascot project
- **RAG-OA**: Document analysis with RAG
- **storage-explorer-extension**: Cloud storage management

### Day 5 - Advanced Tools & CI/CD
- **CI-COSC**: Continuous Integration/Deployment pipeline
- **Duplicate-Image-Finder**: Image deduplication tool
- **Image-Compression-Analyzer**: Image optimization
- **human-vs-ai-detector**: AI content detection

## 🚀 Getting Started

Each project has its own README with specific setup instructions. To get started:

1. Navigate to the day you're interested in: `cd dayX/`
2. Choose a project folder: `cd project-name/`
3. Read the project-specific README for setup instructions

## 🛠️ Technologies Covered

- **Web Development**: HTML, CSS, JavaScript, React, Vue
- **Backend**: Python (FastAPI, Flask), Node.js
- **AI/ML**: Ollama, TensorFlow, PyTorch, OpenAI
- **DevOps**: Docker, CI/CD, GitHub Actions
- **Security**: Vulnerability scanning, secret detection
- **Databases**: Vector stores, SQL, NoSQL
- **Browser Extensions**: Chrome Extension APIs

## 📋 Prerequisites

General requirements across projects:
- Node.js 16+ for JavaScript projects
- Python 3.8+ for Python projects
- Docker for containerized applications
- Git for version control

Specific prerequisites are listed in individual project READMEs.
