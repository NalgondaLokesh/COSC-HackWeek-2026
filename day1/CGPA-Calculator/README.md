# CGPA Calculator

A simple web-based CGPA calculator built with HTML, CSS, and JavaScript.

## Features
- Calculate CGPA from subject grades and credits
- Simple and responsive interface
- Easy to use in a browser

## Usage
1. Open index.html in your browser.
2. Enter the required course details.
3. Click the calculate button to view the result.

- index.html - Main application file
