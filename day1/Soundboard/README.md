# Soundboard

A simple browser-based soundboard app built with a single `index.html` file.

## Features

- 12 sound buttons
- Volume control
- Stop all sounds
- No external audio files required
- Built using only HTML, CSS, and client-side JavaScript/Web Audio API

## Usage

1. Open `index.html` in a modern web browser.
2. Use the slider to adjust volume.
3. Click any sound button to play a sound.
4. Click `Stop All` to stop all playing sounds.

## Notes

- The app is designed for local use.
- Works best in browsers with Web Audio API support.
