# Kanban Board

A simple Trello-style Kanban board built with plain HTML, CSS, and JavaScript. It supports:

- Creating cards in three columns: To Do, In Progress, and Done
- Dragging cards between columns
- Deleting cards
- Persisting board data in localStorage

## Run locally

Open [index.html](index.html) in your browser to use the app.

## Project structure

- [index.html](index.html) — main app UI and logic

## Notes

The board stores changes locally in the browser, so data persists on the same device/browser.
