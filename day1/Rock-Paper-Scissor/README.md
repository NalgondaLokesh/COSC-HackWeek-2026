# Rock · Paper · Scissors

A simple browser-based Rock, Paper, Scissors game built with a single `index.html` file.

## Features

- Player vs computer gameplay
- First to 5 wins
- Scoreboard updates for player, computer, and round count
- Result display with move summary and winner message
- Reset game and reset scores controls
- Keyboard shortcuts: `1` = Rock, `2` = Paper, `3` = Scissors, `R` = Reset

## How to Run

1. Open `index.html` in your browser.
2. Click one of the move buttons: Rock, Paper, or Scissors.
3. Play until either the player or computer reaches 5 wins.
4. Use `Reset Game` to restart from zero.

## Notes

- The game runs entirely in the browser with no external dependencies.
- Refreshing the page also resets the game state.

- `index.html` — game UI and logic in one file
