# 🤖 Ollama Chat

A beautiful chat application that interacts with local Ollama models. Built with vanilla JavaScript, Express, and the Ollama API.

## ✨ Features

- 💬 Real-time chat with Ollama models
- 📝 Message history with localStorage persistence
- 🔄 Loading indicators for responses
- 🗑️ Clear chat functionality
- 📦 Model switching (supports multiple models)
- 🎨 Beautiful dark theme UI
- ⌨️ Keyboard shortcuts (Ctrl+Enter to send)
- 📱 Responsive design

## 🛠️ Requirements

- [Ollama](https://ollama.ai/) installed and running
- At least one model pulled (e.g., `ollama pull llama2`)
