# Storage Explorer - Browser Extension

## 🗄️ Overview
A powerful browser extension that allows you to explore and manage client-side storage (localStorage, sessionStorage, and cookies) of any website with a clean and intuitive interface.

## ✨ Features

- **View All Storage**: Display localStorage, sessionStorage, and cookies in one unified interface
- **Search Functionality**: Quickly find specific entries by key or value
- **Filter by Type**: Filter storage items by type (Local, Session, Cookies)
- **View Details**: See key, value, storage type, and approximate size for each item
- **Delete Individual Items**: Remove specific storage entries
- **Clear All Storage**: Remove all storage data for the current website
- **Export/Import**: Backup and restore storage data as JSON
- **Real-time Updates**: Refresh data with a single click
