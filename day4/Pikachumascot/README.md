# Pikachu Mascot

A simple pure-CSS illustration of Pikachu built with HTML and CSS.

## What it is

This project renders a stylized Pikachu mascot entirely using HTML elements and CSS styling, with no external images or libraries.

## Run locally

1. Open [index.html](index.html) in your browser.
2. Alternatively, serve the folder with a simple static server if you prefer.
