# Color Blindness Simulator

A simple web-based simulator that lets users upload an image and preview how it appears under common color vision deficiencies such as protanopia, deuteranopia, and tritanopia.

## Features

- Upload an image from your device
- Switch between different color blindness simulation modes
- Preview the transformed image instantly in the browser
- Clean, responsive UI for quick experimentation

## How to Run

Open [index.html](index.html) in a browser to use the simulator.
