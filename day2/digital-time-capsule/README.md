# Digital Time Capsule

A polished single-page web app for creating time capsules that unlock on a future date.

## Features
- Create capsules with a title, unlock date, message, privacy setting, and optional image
- View live countdowns for locked capsules
- Toggle between light and dark themes
- Store capsules locally in the browser
- Modern, responsive UI with a cleaner card-based experience

## Run locally
1. Open the project folder in a browser, or serve it with a simple static server.
2. For example, run:
   ```bash
   python -m http.server 8000
   ```
3. Visit http://localhost:8000
