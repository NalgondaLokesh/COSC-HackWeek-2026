# Day 2

## Projects

### Color-Blindness-Simulator
A web application that simulates different types of color blindness to help understand visual accessibility challenges.

### Digital-Business-Card-Generator
A tool for creating and customizing digital business cards with various design options.

### File-Metadata-Inspector
An application for inspecting and analyzing file metadata and properties.

### digital-time-capsule
A digital time capsule application for storing and retrieving messages or content for future dates.

### habit-streak-tracker
A productivity tool for tracking daily habits and maintaining streaks.

### white-board
A collaborative whiteboard application for drawing and brainstorming.

## Technologies
- HTML, CSS, JavaScript
- Canvas API for drawing applications
- File API for file handling
- Local storage and IndexedDB
- Modern JavaScript frameworks (React/Vue where applicable)

Each project has its own README with specific setup instructions. Navigate to individual project folders for detailed documentation.
